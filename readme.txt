Pickle Post to Twitter

== Description ==

Send posts to a twitter account/feed.

== Installation ==

1. Upload `pickle-post-to-twitter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
